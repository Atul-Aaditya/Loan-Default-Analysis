# 📊 Loan Default Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an end-to-end Power BI dashboard built to analyze loan default patterns and identify the key factors influencing loan repayment behavior. The dashboard helps financial institutions monitor applicant profiles, evaluate financial risks, and gain insights into default trends for better decision-making.

---

## 🎯 Objectives

- Analyze loan default trends
- Identify high-risk applicants
- Monitor applicant demographics
- Understand financial indicators affecting defaults
- Support data-driven lending decisions

---

## 📈 Dashboard Pages

### 1. Loan Default & Overview
- Overall loan performance

### 2. Applicant Demographics & Financial Profile
